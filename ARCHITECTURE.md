## üßÆ Decimal Calculations Boundary

The decimal calculation layer (`src/utils/calculations-decimal.ts`) defines a lightweight `DecimalBudget` interface for number‚Äëcrucial math using `decimal.js`.

- Field mapping: `DecimalBudget.category` intentionally holds the categoryId string. This keeps the calculation layer framework‚Äëagnostic and avoids coupling it to wider domain models.
- Call‚Äësite contract: When calling decimal helpers, map domain `budget.categoryId` ‚Üí decimal `category` (e.g., `{ ...budget, category: budget.categoryId }`). Resolve human‚Äëreadable names via the `categories` list when needed for display.
- Documentation: This contract is also documented inline in `src/types/decimal-types.ts`.

This approach minimizes refactors and maintains clear boundaries between domain models and calculation utilities.
